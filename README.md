# CodeAlpha_SQL_Injection_Prevention_System
 A Python-based security system that prevents SQL Injection attacks by using parameterized SQL queries. It securely stores user credentials in an SQLite database and provides safe user registration and login functionality, helping protect sensitive data from unauthorized access and common SQL injection vulnerabilities.
# SQL Injection Prevention System

## Description
This project is developed using Python and SQLite to protect user data from SQL Injection attacks. It uses parameterized queries to securely handle user inputs and prevent unauthorized database access.

## Features
- User Registration
- Secure User Login
- SQLite Database
- SQL Injection Prevention
- Parameterized Queries

## Technologies Used
- Python
- SQLite3

## How to Run
1. Install Python
2. Run:
   python main.py
